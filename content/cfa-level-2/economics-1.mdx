## Economics

### Capital Market expectations

- Benefits:
  1.  Helps decide stategic asset allocation (SAA)
  2.  Helps detect short-term mispricings through tactical asset allocation (TAA)
  3.  Used in equity/bond valuation (beta research)

#### Process

- Assess drivers of past performance
- Choose valuation model
- Evaluate data to be used
- Formulate expectations using Black-Litterman, DCF, etc.
- Monitor performance and refine process

#### Weaknesses

- Data measurement errors and bias
- Use of ex-post data
- Non-repeating patterns
- Failure to incorporate conditioning information
- Misinterpretation of correlations
- Psychological traps
- Model and input uncertainty

### Forecasting Tools

1. Statistical tools
2. DCF Models
3. Financial Equilibrium Models (e.g., ICAPM)
4. Risk Premium Appoach

#### Statistical Tools

1.  Project historical data

- The variance of market i, given a covariance of factor returns:

<Math>
  {
    "$\\sigma_{i}^2 = \\beta_{i,1}^2\\sigma_{F,1}^2 + \\beta_{i,2}^2\\sigma_{F,2}^2 + 2\\beta_{i,1}\\beta_{i,2}Cov(F_{1},F_{2}) + \\sigma_{\\varepsilon,i}^2$"
  }
</Math>

- The covariance between two markets i and j:

<Math>
  {
    "$Cov(i,j) = \\beta_{i,1}\\beta_{j,1}\\sigma_{F,1}^2 + \\beta_{i,2}\\beta_{j,2}\\sigma_{F,2}^2 + (\\beta_{i,1}\\beta_{j,2} + \\beta_{i,2}\\beta_{j,1})Cov(F_{1},F_{2})$"
  }
</Math>
